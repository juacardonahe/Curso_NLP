# Curso de NLP — Procesamiento de Lenguaje Natural

¡Bienvenido/a a este repositorio de NLP (Natural Language Processing)!  
Aquí encontrarás proyectos, ejercicios y recursos prácticos realizados durante el curso.

## Contenido 📚
- 📄 Fundamentos de NLP
- ✨ Preprocesamiento de texto
- 🛠️ Técnicas de tokenización, stemming y lematización
- 🔥 Modelado de lenguaje y embeddings
- 🤖 Modelos de clasificación de texto
- 🧠 Redes neuronales aplicadas al lenguaje natural
- 📈 Análisis de sentimientos y generación de texto
- 🗂️ Mini-proyectos y casos prácticos

## Tecnologías y Herramientas 🚀
- Python 3.10+
- NLTK, spaCy, Scikit-learn
- Hugging Face Transformers
- TensorFlow / PyTorch
- Jupyter Notebooks
- Git & GitHub

## Organización del Repositorio 🗂️
- `notebooks/` — Notebooks de prácticas y teoría
- `src/` — Código fuente de proyectos y funciones auxiliares
- `data/` — Datasets utilizados (o enlaces a los datasets)
